🔌 How This Integration Works

This demo store uses the VeilReceipt Checkout Widget to accept private payments. Here's how any e-commerce platform can integrate:

1

Get API Key

Sign up as a merchant on VeilReceipt and generate an API key from the Integration dashboard.

2

Create Session

When a customer checks out, your backend creates a payment session via the API.

3

Redirect to Pay

Redirect to the VeilReceipt checkout page or open it in a popup window.

4

Receive Webhook

Get notified via webhook when payment is confirmed with a ZK proof.

Backend: Create Payment Session

// Node.js / Express example app.post('/checkout', async (req, res) => { const response = await fetch('https://veilreceipt-api.onrender.com/integrate/payments', { method: 'POST', headers: { 'Content-Type': 'application/json', 'X-API-Key': process.env.VEIL_API_KEY, }, body: JSON.stringify({ amount: req.body.total, currency: 'credits', description: `Order #${req.body.orderId}`, redirect_url: `https://mystore.com/order/${req.body.orderId}/success`, metadata: { orderId: req.body.orderId } }), }); const { checkout_url } = await response.json(); res.redirect(checkout_url); });

Frontend: Embeddable Widget

<!-- Add to your HTML --> <script src="https://veil-receipt.vercel.app/veil-checkout.js"></script> <script> // Open checkout in a popup VeilCheckout.open({ sessionId: 'ps_abc123', mode: 'popup', onSuccess: function(data) { console.log('Payment confirmed!', data.tx_id); } }); </script>

Shopify / WooCommerce

// Shopify App — in your checkout extension: const session = await createVeilPaymentSession({ amount: checkout.totalPrice, currency: 'credits', metadata: { shopify_order_id: checkout.orderId }, redirect_url: checkout.thankYouUrl, }); // Redirect customer to VeilReceipt checkout return { redirect: session.checkout_url }; // WooCommerce — in your payment gateway plugin: // POST to /integrate/payments from your PHP backend // Then redirect: wp_redirect($response['checkout_url']);
